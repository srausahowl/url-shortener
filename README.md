# URL Shortener

Client-side URL shortener that encodes the target into a base64 hash fragment and redirects on load. No backend, no tracking.

## Live Demo

https://srausahowl.github.io/url-shortener/

## Usage

Open `index.html` in any modern browser. Zero dependencies, zero build step.

```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```

## Tech

- Vanilla HTML / CSS / JavaScript
- No frameworks, no build tools
